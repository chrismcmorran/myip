# myip --- A utility for quickly checking your WAN/LAN addresses

### Requirements
Python 2.7.x

### Install
mv myip.py /usr/local/bin/myip

sudo chmod +x /usr/local/bin/myip

### Usage
#### To display both your LAN and WAN
myip
#### To display only your LAN
myip -l
#### To display only your WAN
myip -w
